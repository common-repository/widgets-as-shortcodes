**Insert WordPress default widgets Anywhere in your site**

Basically WordPress default widgets can be inserted anywhere in your site:

* Page/post WYSIWYG editor by using the buttion in the editor
* In sidebars by using the default Widgets themselves
* In template files by using the WordPress default widgets' shortcode and WordPress do_shortcode function

**Custom styling**

If you need to further style an widget you can use the default WordPress classes for those widgets.
